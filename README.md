## Features

- **High Performance**: Uses HTML5 Canvas API for smooth 60FPS gameplay.
- **Modern Tech Stack**: Built with TypeScript for reliability and Vite for speed.
- **Neon Aesthetic**: Cyberpunk-inspired visuals with glowing effects.
- **Particle Effects**: Satisfying explosions when collecting food.
- **Responsive**: scales to fit the screen (work in progress).

## Getting Started

### Prerequisites

- Node.js (v14 or higher)

### Installation

1.  Clone the repository:

    ```bash
    git clone https://github.com/bagaskara112/Snake-Game.git
    cd Snake-Game
    ```

2.  Install dependencies:

    ```bash
    npm install
    ```

3.  Run the development server:

    ```bash
    npm run dev
    ```

4.  Open your browser and visit the URL shown in the terminal (usually `http://localhost:5173`).

## Controls

- **Arrow Keys** or **WASD**: Move the snake
- **Spacebar**: Start / Restart Game
